# CONTRIBUTING
## Development Setup
### Prerequisites
* Install [Node.js](https://nodejs.org/) ([version](../.node-version)) which includes [Node Package Manager](https://docs.npmjs.com/)

### Setting Up
1. git clone this repository
1. `npm ci`
